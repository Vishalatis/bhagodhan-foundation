Immediate Opening For React Js Developer

Job Description: We are seeking a talented React.js Developer to join our dynamic team. The ideal candidate will have hands-on experience with React.js and a passion for building interactive, responsive, and high-performing user interfaces.

  • Key Responsibilities:
  • Develop and maintain React.js applications with clean and scalable code.
  • Collaborate with designers, backend developers, and other team members to deliver high-quality solutions.
  • Ensure the technical feasibility of UI/UX designs.
  • Optimize components for maximum performance across various devices and browsers.
  • Troubleshoot and debug application issues in a timely manner.
  • Requirements:
  • Proficient in React.js and its core principles.
  • Strong understanding of JavaScript, ES6+, HTML, and CSS.
  • Experience with Redux or Context API for state management.
  • Familiarity with RESTful APIs and integrating backend services.
  • Knowledge of modern front-end build pipelines and tools such as Webpack, Babel, etc.
  • Preferred Skills:
  • Familiarity with Git and version control.
  • Knowledge of unit testing with tools like Jest or Enzyme.
  • Experience in Agile development practices.
